Flipzy Local Site Package
=========================

Files included:
- index.html
- assets/logo.png
- assets/fox.png
- assets/profile-picture.png
- assets/youtube-banner.png

How to run locally:
1. Unzip the folder.
2. Open index.html in your browser.

Notes:
- This version is fully static HTML/CSS and does not require React, Node, or a build step.
- The waitlist form is a demo only. It shows a confirmation alert locally.
- If you want, the next step is connecting the form to Mailchimp, ConvertKit, Brevo, or Formspree.
